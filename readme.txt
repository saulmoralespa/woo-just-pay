Pasarela (Gateway) de pago de Chile

== Description ==

Pasarela (Gateway) de pago enfocada en la reducción de la fricción del proceso de compra, mejora de conversión de ventas y control del riesgo de fraude.

== Installation ==

1. Download the plugin
2. Enter the administrator of your WordPress.
3. Enter Plugins / Add-New / Upload-Plugin.
4. Find the plugin downloaded on your computer and upload it like any other file.
5. After installing the .zip you can see it in the list of installed plugins, you can activate or deactivate it.
6. To configure the plugin you must go to: WooCommerce / settings / Payu Latam / manage
